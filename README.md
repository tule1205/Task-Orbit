# TaskOrbit

A beautiful, fully-featured Kanban-style task board where users can create tasks, drag them across board sections, and manage work visually.

## Deploy on Vercel

1. Import this repo in [Vercel](https://vercel.com).
2. Set **Root Directory** to `Frontend` (Project Settings → General).
3. Leave **Build Command** empty; **Output Directory** is the root (`.`).
4. Deploy. The app will be served at `/`, `/inprogress`, `/inreview`, `/done`.
